# to_do_by_python
This is a simple command-line To-Do List application written in Python.
It allows users to add, view, and delete tasks using a menu-driven program.
This project is ideal for Python beginners
WHAT THIS CODE WILL DO :-
          Add a new task
          Show all tasks
          Delete a task by its number
          Exit the program safely
HOW THIS PROGRAM WORK :-
          Tasks are stored in a Python list
          The program runs in a loop until the user exits
          The user selects an option from the menu:
          1 → Add task
          2 → View tasks
          3 → Delete task
          4 → Exit
